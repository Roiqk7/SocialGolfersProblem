Social Golfer's Problem Solver

Can N golfers be scheduled to play in G groups of size S over R rounds so that no pair of golfers plays together more than T times?

Schedule

About this SGP solver

On the backend the application encodes the problem into DIMACS CNF and using the Glucose SAT solver finds the solution.
